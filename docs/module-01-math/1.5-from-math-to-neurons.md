## What is a Neuron?

In an AI, a [**neuron**](../glossary.md#neuron) is just a mathematical function that takes some inputs, does a calculation, and spits out an output. A **Layer** is just a collection of these neurons working together.

Every layer in a neural network follows a simple three-step process:

![Neuron Math](diagrams/neuron.svg)

1.  [**Weights ($W$)**](../glossary.md#weights-and-biases): Multiply the input vector by a weight matrix (Matrix-Vector Multiplication).
2.  [**Bias ($b$)**](../glossary.md#weights-and-biases): Add a small "nudge" to the result (Vector Addition).
3.  [**Activation Function ($f$)**](../glossary.md#activation-function): Pass the final result through a simple "decision" function.

The whole thing can be written as one elegant formula:
$$y = f(Wx + b)$$

## Step 1: Weights ($W$)

The [**Weights**](../glossary.md#weights-and-biases) are the most important part. They represent the "knowledge" of the model. By multiplying the input vector by the weight matrix, the AI transforms the data into something more meaningful.

## Step 2: Bias ($b$)

Sometimes, even if the weights are perfect, the result needs a little adjustment. The [**Bias**](../glossary.md#weights-and-biases) is just a vector that we add to the result. It allows the model to be more flexible.

## Step 3: Activation Functions ($f$)

This is where the magic happens. If we *only* used weights and biases, our AI would just be a series of simple linear transformations. But real-world problems are complex!

An [**Activation Function**](../glossary.md#activation-function) is a simple rule that adds "non-linearity." The most common one in AI is called **ReLU** (Rectified Linear Unit). Its rule is simple: **If a number is negative, turn it into 0. If it's positive, keep it as it is.**

$$ReLU(x) = \max(0, x)$$

## Building a Neuron Layer in Python

Let's build a tiny "layer" with 3 neurons that takes 3 inputs.

```python
import numpy as np

# Step 0: Our input vector (e.g., three properties of a word)
x = np.array([1.5, -2.0, 0.5])

# Step 1: Weight Matrix (learned by the AI over time)
W = np.array([
    [0.1, 0.5, -0.1],
    [0.2, -0.3, 0.8],
    [0.9, 0.1, -0.4]
])

# Step 2: Bias Vector (also learned by the AI)
b = np.array([0.1, 0.2, 0.1])

# Step 3: Perform the calculation
# Multiply weights by input and add bias
linear_step = (W @ x) + b

# Step 4: Apply the Activation Function (ReLU)
# np.maximum(0, array) compares every element to 0
y = np.maximum(0, linear_step)

print(f"Input Vector (x): {x}")
print(f"Linear Result (Wx + b): {linear_step}")
print(f"Final Neuron Output (y): {y}")
```

Running this prints:

```text
Input Vector (x): [ 1.5 -2.   0.5]
Linear Result (Wx + b): [-0.8   1.5   1.05]
Final Neuron Output (y): [0.   1.5  1.05]
```

In this example, the neurons transformed the input `x` into the output `y`. This is the fundamental loop of *every* AI model on the planet!

import numpy as np

x = np.array([2.0, -1.0])

W = np.array([

    [-0.2, 0.8]
])

# Bias vector (2 outputs)
b = np.array([0.1, -0.1])

# Forward pass
z = W @ x + b

**Exercise 2:** Why is the ReLU activation function important? What happens if we remove it and only use linear transformations?

<details>
<summary>Show solution</summary>

ReLU adds non-linearity, allowing neural networks to learn complex patterns. Without it:

```python
import numpy as np

x = np.array([1.0, 2.0])
W1 = np.array([[1.0, 2.0], [3.0, 4.0]])
b1 = np.array([0.5, 1.0])
W2 = np.array([[2.0, 3.0], [4.0, 5.0]])
b2 = np.array([0.2, 0.3])

# Without activation (just linear)
z1 = W1 @ x + b1
z2 = W2 @ z1 + b2

print(f"Two linear layers result: {z2}")
print("This is still just a linear transformation!")
print(f"It's equivalent to: W2 @ W1 @ x + (W2 @ b1 + b2)")
```

Without ReLU, stacking multiple layers is mathematically equivalent to having just one layer, so the network cannot learn complex patterns.
