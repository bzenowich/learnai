[RAG](5.2-rag.md) is perfect for searching through documents. But what if the information an AI needs is constantly changing?

If you ask an AI "What is the current temperature in London?", it can't find that in a PDF from last year. It needs **Dynamic Context**.

## How AI Uses Tools

To access real-time information, an AI model uses a technique called **Tool Calling**. 

1.  **Tool Definitions:** The AI is given a list of available "tools" (APIs) it can use.
    *   Example: `get_weather(city)`, `search_google(query)`, `calculate(expression)`.
2.  **The Thought Loop:** When the AI reads your question, it decides if it needs a tool to answer.
3.  **The Call:** Instead of generating an answer, the AI generates a special "command" to call the tool.
4.  **The Execution:** A separate program (the "orchestrator") runs the tool and gets the result (e.g., "72 degrees").
5.  **The Integration:** The tool's output is pasted into the AI's **[Context Window](../glossary.md#context-window)** (just like RAG!).
6.  **The Final Answer:** Now, with the real-time data, the AI can answer your question accurately.

## Visualizing Tool Calling in Python

Let's imagine our AI has a `get_weather` tool.

```python
import datetime

def get_current_time():
    # A simple tool that gets the current time
    return datetime.datetime.now().strftime("%H:%M:%S")

def ai_brain(user_query):
    # 1. The AI decides if it needs a tool
    if "time" in user_query.lower():
        print("[AI Thought]: I need to know the current time.")
        
        # 2. Call the tool
        time_data = get_current_time()
        
        # 3. Augment the context!
        context = f"The current time is {time_data}."
        
        # 4. Generate the final answer
        return f"Based on my tools, {context}"
    else:
        return "I'm not sure what you're asking."

# Test our tool-using AI!
print(ai_brain("What time is it right now?"))
```

Running this prints:
```text
[AI Thought]: I need to know the current time.
Based on my tools, The current time is 15:08:14.
```

## Why this is Revolutionary

By giving an AI access to tools, we are turning it from a simple "Chatbot" into an **[Agent](../glossary.md#agent)**. 

An agent can:
*   Search the web for news.
*   Check your calendar.
*   Calculate complex math.
*   Read and write files on your computer.

But there's a problem: every AI has a different way of "talking" to tools. To fix this, a new standard was created called **[MCP](../glossary.md#model-context-protocol)**.

**1. Why does the tool calling approach use the Context Window instead of just returning the answer directly?**

The tool returns raw data (e.g., "15:08:14"), not a natural language response. By pasting this data into the Context Window alongside the original question, the LLM can format, interpret, and contextualize the data appropriately. This leverages the LLM's strength (language generation) while using tools for what they're best at (data retrieval).

**2. What's the difference between RAG and Tool Calling?**

**RAG** is for accessing *static, searchable documents* (books, articles, wikis). **Tool Calling** is for triggering *dynamic actions* (getting the time, calling an API, running code, querying a live database). Both augment the Context Window with external information, but tool calling is triggered by the AI's decision ("I need to call this tool"), while RAG is typically triggered by a semantic search.

import datetime

def get_current_time():
    return datetime.datetime.now().strftime("%H:%M:%S")

def get_weather(city):
    # Simulated weather data
    weather_data = {
        "London": "Cloudy, 15°C",

def ai_brain(user_query):
    if "time" in user_query.lower():

        time_data = get_current_time()

        return f"Based on my tools, {context}"
    elif "weather" in user_query.lower():
        print("[AI Thought]: I need to check the weather.")
        city = "London"  # simplified extraction
        weather = get_weather(city)
        context = f"The weather in {city} is {weather}."
        return f"Based on my tools, {context}"
    else:
        return "I'm not sure what you're asking."

print(ai_brain("What is the weather in London?"))
