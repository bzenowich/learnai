# 6.4 Level 4: Agentic AI

Level 3 models are amazing thinkers. But an AI that can only "think" is like a person with a map but no car. They know where to go, but they can't actually get there. 

This brings us to **Level 4: Agentic AI**.

## What is a Level 4 AI?

A Level 4 AI is an **Agent**. It combines the reasoning of Level 3 with the tools of Module 5 to take autonomous actions in the real world. 

Instead of just answering your question, the AI can:
1.  **Plan:** Break a complex task into multiple sub-steps.
2.  **Execute:** Use tools (like MCP, Module 5.4) to perform actions like searching the web, reading files, or writing code.
3.  **Reflect:** Look at the result of its actions and decide if it needs to try again or move to the next step.

### The Agentic Loop

Imagine you ask an AI: "Research the current price of Bitcoin and write a Python script that calculates how much $1,000 would buy today."

1.  **Plan:** "I need to search for the current Bitcoin price, then write a Python script."
2.  **Act:** The AI calls a `google_search` tool.
3.  **Result:** It sees the price is $60,000.
4.  **Reflect:** "Now that I have the price, I can write the script."
5.  **Act:** The AI writes the Python code.
6.  **Deliver:** "Here is your script! $1,000 will buy you 0.016 Bitcoin today."

## Visualizing Level 4 in Python

```python
# A simple autonomous agent loop
def level_4_agent(user_task):
    print(f"[Plan]: {user_task}")
    
    # 1. Step one: Research
    print("[Action]: Researching current data via Tool...")
    current_data = {"price": 50000} # Simulated tool result
    
    # 2. Step two: Thinking / Processing
    print(f"[Reflection]: Okay, price is {current_data['price']}. Now I can do the math.")
    
    # 3. Step three: Action (like writing a file)
    print("[Action]: Writing Python script to 'bitcoin_calc.py'...")
    
    # 4. Final delivery
    return "Task Complete: The script has been written and verified."

print(level_4_agent("Calculate my Bitcoin buy."))
```

## Why this is Revolutionary

Agentic AI is a fundamental shift. We are no longer just "chatting" with a model; we are **delegating** work to it. 

Level 4 agents can act as your personal researcher, your junior coder, or even your administrative assistant.
